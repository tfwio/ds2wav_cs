


Console app to generate 16-bit mono 44.1 kb/s PCM R-IFF/WAVE from *.ds programs.


**use**

```
ds2wav [ <directory> | <*.ds> ]
```

Where `path` is either a `*.ds` program (file) or a directory containing
multiple `*.ds` programs.
